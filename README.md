## 🎯 Overview

This project performs exploratory data analysis (EDA) and machine learning on Spotify music data to:
- Analyze audio features and their correlations
- Cluster songs based on danceability, energy, and other characteristics
- Build a recommendation system to suggest similar tracks
- Visualize music patterns and genre distributions

## ✨ Features

- **Data Preprocessing**: Clean and prepare Spotify dataset with 32,000+ unique tracks
- **Exploratory Data Analysis**: 
  - Statistical summaries of audio features
  - Correlation matrix visualization
  - Genre and cluster distribution analysis
- **Machine Learning**:
  - K-Means clustering with optimal k selection (Elbow Method)
  - PCA dimensionality reduction for visualization
  - K-Nearest Neighbors recommendation system
- **Interactive Visualizations**:
  - Heatmaps for feature correlations
  - Scatter plots for cluster visualization
  - Genre distribution charts
